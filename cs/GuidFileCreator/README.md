# GuidFileCreator

This is a .NET 10.0 console application.

## Getting Started

1. Ensure you have .NET 10.0 SDK installed.
2. Build and run the app:
   - `dotnet build`
   - `dotnet run`

## Tips

"window.newWindowDimensions": "maximized"
inside .vscode/settings.json can be used to maximize Visual Studio Code.

## Project Structure
- Program.cs: Main entry point
- GuidFileCreator.csproj: Project file

## Purpose
This app is a starting point for creating files with GUIDs or other prototyping tasks.
